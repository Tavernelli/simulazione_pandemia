# Simulazione Pandemia

Applicazione Java per simulare la diffusione di una pandemia in una popolazione.

## Descrizione

Il progetto mostra una simulazione grafica in cui le persone si muovono in una finestra e possono infettarsi a vicenda se si avvicinano.

## Funzionalità principali

- Simulazione di persone con stati: suscettibile, infetto, asintomatico, immune e morto
- Controllo della popolazione iniziale, percentuale di infetti, infettività, asintomatici, mortalità e immunità
- Selezione area per vaccinazione o quarantena
- Animazione grafica e logica di contagio con distanza minima

## Esecuzione

La classe principale è `simulazionepandemia.SimulazionePandemia`.

Esegui il progetto da un ambiente Java compatibile con Swing.

## Note

Il progetto è strutturato come applicazione NetBeans con la cartella `nbproject` e i sorgenti in `SimulazionePandemia/src/simulazionepandemia`.
